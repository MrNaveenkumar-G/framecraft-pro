# 🖼️ FrameCraft Pro — Photo Frames Business Website

A complete, professional **Photo Frames Business Website** with Invoice Management, Dashboard, Customer CRM, Reminders, and more. Built with **HTML, Bootstrap 5, JavaScript & CSS**. Deployable for **FREE** on GitHub Pages.

---

## 🚀 Live Demo Setup (GitHub Pages — FREE URL)

### Step 1: Create GitHub Account
Go to [github.com](https://github.com) and sign up for free.

### Step 2: Create New Repository
1. Click the **"+"** button → **"New repository"**
2. Repository name: `framecraft-pro` (or any name you like)
3. Set to **Public**
4. Click **"Create repository"**

### Step 3: Upload Files
1. Click **"uploading an existing file"** link
2. Drag & drop ALL the project files and folders:
   - `index.html`
   - `dashboard.html`
   - `invoice.html`
   - `invoice-history.html`
   - `reminders.html`
   - `customers.html`
   - `products.html`
   - `settings.html`
   - `about.html`
   - `contact.html`
   - `css/` folder (with `style.css`)
   - `js/` folder (with all `.js` files)
3. Click **"Commit changes"**

### Step 4: Enable GitHub Pages
1. Go to repository **Settings** tab
2. Scroll to **"Pages"** section (left sidebar)
3. Under **Source**, select **"Deploy from a branch"**
4. Branch: **main** → Folder: **/ (root)**
5. Click **Save**

### Step 5: Your FREE URL is Ready! 🎉
After 1–2 minutes, your website will be live at:
```
https://YOUR-USERNAME.github.io/framecraft-pro/
```

## ✨ Features

### 🏠 Public Pages
- **Home** — Hero, featured products, testimonials, quick actions
- **Products** — Full catalog with filters, stock tracker
- **About** — Company story, team, values
- **Contact** — Contact form, WhatsApp order, business hours

### 📊 Business Dashboard
- Revenue stats with animated counters
- Monthly revenue chart (Chart.js)
- Invoice status donut chart
- Recent invoices table
- Stock overview for all products
- Upcoming reminders widget

### 🧾 Invoice System
- Create professional invoices instantly
- Customer autocomplete search
- Quick add products from catalog
- GST toggle (5%, 12%, 18%, 28%)
- Discount percentage field
- Print-ready invoice with beautiful layout
- **WhatsApp share** invoice summary
- Save as PDF via browser print
- Export invoice history as CSV

### 📋 Invoice History
- All invoices with search & filter
- Filter by status, date range, customer
- Update payment status inline
- View full invoice details in modal
- Print any past invoice
- Share via WhatsApp
- Delete invoices
- Export all to CSV

### 🔔 Reminders
- Add reminders with priority levels (Urgent / Today / Upcoming)
- Types: Payment, Delivery, Stock Order, Customer Call, Admin
- Mark as done
- Filter by priority
- Reminder count badge in navbar

### 👤 Customer Management
- Add / Edit / Delete customers
- Quick invoice creation from customer card
- Total orders & spending per customer
- Search customers

### 🎨 User Customization (Settings)
- Upload profile/avatar photo
- Change business name, phone, email, address, GST
- Social media links (WhatsApp, Instagram, Facebook)
- **Light / Dark mode** toggle
- **6 accent color themes** + custom color picker
- Invoice default notes
- Export all data as JSON backup
- Reset all data option

## 📞 Customization

### Change Business Name & Details
1. Open the website → Go to **Settings** page
2. Update Business Name, Phone, Email, Address, GST
3. Click **Save Changes**
→ All pages update automatically!

### Change Logo Color
1. Settings → Pick any accent color
2. Entire website updates instantly

### Add Products
1. Go to **Products** page
2. Click **"Add Product"**
3. Fill name, size, price, stock, category
